# TSA State Video Game Design 1st Place Entry
A short project coded in two weeks for the TSA State competition.

Unfortunately, it will not be able to compete in nationals, but I hope this helps anyone who's looking for an example game (since I felt a severe lack of resources when competing in this category).

All the art and music were bought and sourced from itch, so please do not take any individual assets out for personal use. Very heavy inspiration for this project comes from this comprehensive guide: https://dev.to/christinec_dev/lets-learn-godot-4-by-making-an-rpg-part-1-project-overview-setup-bgc

Here is the game on itch: https://idle-lily.itch.io/cat-quest
Here is the video tutorial: https://www.youtube.com/watch?v=0ZNQMcPU_bE&feature=youtu.be
